00 · Shape language — the new geometry vocabulary

The Cyberpunk Blue surface previously read as "soft squared" — 2–3px radii. The new shape language replaces that with chamfered polygons built from clip-path. The signature silhouette is an asymmetric cut on the top-right + bottom-left corners that reads as forward-leaning, interactive. The colour palette and typography stay unchanged — only the silhouette is new.

01 · Shape catalog

TR-BL · default
Top-right + bottom-left cut. The signature interactive silhouette. Used on buttons, inputs, panels.
TL-BR · mirrored
Mirrored chamfer. Pairs with default to form nose-to-nose chevrons. Use on "passive" / readout blocks.
HEX · all four cut
Symmetric. Reads as a data tile / readout plate. Use on stat values, dice readouts, status badges.
CARD · top corners
Cut at top only — bottom edge flat so cards stack. Use for content cards inside scroll regions.
PANEL · stepped TL
Big TR + BL chamfer, small step at top-left. The "main sheet" silhouette. Use once per screen.
NOTCH · single corner
One TR cut, three square edges. HUD callout / alert slab. Use sparingly — high-energy.
TAG · chevron tip
Pointed left edge, flat right. Status flags, faction markers. Read as "directional".
ARROW · forward action
Right-pointing chevron block. Primary action buttons (Commit, Roll, Continue).

03 · Frame mechanics — how the 1px stroke survives the clip

Problem
A real border gets clipped away by clip-path at the chamfer corners — the stroke disappears precisely where the eye needs it most.
Solution
Paint the outer element in the stroke colour. A ::before pseudo-element inset by 1px paints the inner fill on top, with clip-path: inherit so the inner shape registers with the outer. The 1px gap reads as the hairline.
Hover / focus
Outer background swaps to --cpb-accent. The "border" is just paint, so it animates with a plain background transition. No outline ring (rings get clipped too).
Filled state
On :checked / active, the inner ::before also swaps to the accent. The element reads as a solid plate, no border.
